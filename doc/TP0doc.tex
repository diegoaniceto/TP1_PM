\textbf{\textit \\Resumo: }
\textit{\\ Este relatório descreve a implementação da solução proposta para o o problema de manipulação de e armazenamento de dados de uma rede social de pesquisadores.
  Para que fosse possivel essa implemetação foi utilizado a linguagem Java de programação além de teorias de Orientação a Objeto e Modularização.\\
  O resultado obtido foi satisfatório, tanto em relação a solução do problema, quanto aos conceitos envolvidos.}
